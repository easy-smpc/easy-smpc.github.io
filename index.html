EasySMPC

EasySMPC is an open-source solution to share biomedical data in a privacy-preserving way. You need to install the EasySMPC client and choose between manual, e-mail or dedicated backend-mode. For the latter, you can apply for access to a test server by sending an e-mail or set up your own server.
